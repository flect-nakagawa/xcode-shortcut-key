# xcode-shortcut-key

| 概要            | キー           | 備考           |
| ------------- |-------------|-------------|

| ------------- |-------------|-------------|

| 概要            | キー           | 備考           |
| ------------- |-------------|-------------|

| Show the Assistant Editor | Cmd + Opt + Enter |-| 
| Left Navigator 開閉 | Cmd + 0 |-| 
| Right Navigator 開閉 | Cmd + Opt + 0 |-| 
| Debug Area 開閉 | Cmd + Shift + y |-|
